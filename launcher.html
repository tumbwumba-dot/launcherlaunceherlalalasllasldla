Помощь и поддержка

Быстрый старт

1. Нажмите "Найти GTA 5" для автоматического обнаружения игры

2. Выберите редуксы из каталога

3. Нажмите "Установить" для автоматической установки

4. Запустите игру кнопкой "Запустить с редуксами"

Важная информация

• Перед установкой редуксов создайте резервную копию игры

• Некоторые редуксы могут конфликтовать друг с другом

• Устанавливайте редуксы по одному для избежания проблем

• Проверяйте совместимость редуксов с версией игры

Поддержка

Если у вас возникли проблемы:

• Проверьте логи игры в папке с GTA 5

• Убедитесь что игра закрыта во время установки

• Обратитесь в поддержку через Discord сообщество
